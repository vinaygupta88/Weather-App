## 📌 About The Project

**Weather Web App** is a modern and responsive weather application built using **HTML, CSS, and JavaScript**. It fetches real-time weather information from the **OpenWeatherMap API** and displays current weather conditions along with a **5-Day Forecast**.

The application also includes a **Dark Mode Toggle**, live date & time updates, dynamic weather icons, and a beautiful glassmorphism user interface.

---

## ✨ Features

✔️ Search Weather by City Name

✔️ Real-Time Weather Data

✔️ 5-Day Weather Forecast

✔️ Temperature in Celsius (°C)

✔️ Wind Speed Details

✔️ Error Handling for Invalid Cities

## ⚙️ How To Run

Clone the repository:

```bash
git clone https://github.com/vinaygupta88/weather-app.git

cd weather-app
```

Open:

```text
index.html
```

in your browser.

---

## 🔑 API Configuration

Create your API key from OpenWeatherMap and add it inside your JavaScript file:

```javascript
const apiKey = "YOUR_API_KEY";
```

After adding your API key, save the file and run the application.
